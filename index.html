System Manual & Security Protocol

Follow these steps to initialize QuMail

1 Generate App Password

Go to your Google Account Settings > Security.

Search for App Passwords (Ensure 2-Step Verification is ON).

Select Mail and Other (Custom Name), name it 'QuMail', and copy the 16-character code (remove the spaces) provided.

2 Terminal Configuration

Launch `QuMail.exe`. Navigate to Key Configuration in the left sidebar.

Enter your Gmail ID and paste the 16-character App Password you just generated.

3 Understanding Security Levels

Level 1: KMS OTP

Information-theoretic security. Uses unique cloud entropy for every character.

Level 2: Quantum AES-256

High-speed symmetric encryption using quantum-derived keys.

Level 3: PQC Hybrid

Post-Quantum Cryptography mixed with AES-256 for future-proof security.

Level 4: Sentinel OTP

Maximum protection. Distributed entropy verified by local Sentinel Auditors.

4 Sending & Decrypting

To Send: Go to 'New Message', enter Recipient,Subject, select Security Level,wrtie your message in the 'Message Payload' card and hit 'Send Encrypted'.

To Decrypt: Copy the received cipher-block(Starts with 'DATA:' (copy the entire thing form there till just before "--END--" text) from your inbox, paste it into the 'Inbox' field, and the system will fetch the matching KMS key for local restoration and decrypt it.
